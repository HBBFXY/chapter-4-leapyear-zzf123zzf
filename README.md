# pythonAssignment_leapYear
## 编写程序，对于用户输入的年份，判断其是否为闰年。提示：闰年是能被 4 整除但不能被 100 整除，或者能被 400 整除的年份。
## 要求：使用try except异常语句处理用户输入年份有非数字的情况
